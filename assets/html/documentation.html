Documentation

The documentation of the Varela Digital project describes the conceptual, editorial, and semantic structures adopted to represent the Varela Collection as a scholarly digital edition.

In particular, the project combines TEI XML encoding with a Knowledge Graph based on RDF and JSON-LD in order to support cross-document analysis, interoperability, and the explicit modelling of historical relations.

The following section presents the Data Model, which formalises the core entities and relationships that structure the corpus and enable semantic enrichment beyond individual documents.

In addition to the structural data model, the project defines a domain-specific vocabulary (HRAO) to express historical relations extracted from the corpus.
