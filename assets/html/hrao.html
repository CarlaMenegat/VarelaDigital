Implementation notes

The HRAO ontology documentation is automatically generated from its Turtle declaration using a lightweight Python pipeline based on RDFLib and Jinja2. The approach is conceptually inspired by ontology documentation tools such as Widoco, while remaining fully transparent, reproducible, and project-specific.

caused the death of

URI: https://carlamenegat.github.io/VarelaDigital/hrao#causedDeathOf

Domain: http://xmlns.com/foaf/0.1/Agent

Range: http://xmlns.com/foaf/0.1/Agent

Indicates that the actions of an agent resulted in the death of another agent, as reported in a historical source. This formulation avoids attributing direct or individual intent when the source indicates collective or indirect responsibility.

URI: https://carlamenegat.github.io/VarelaDigital/hrao#participatedInEvent

Domain: http://xmlns.com/foaf/0.1/Agent

Range: http://example.org/san#Event

Indicates that an agent participated in a historical event (e.g., battle, uprising, campaign, meeting), as evidenced by a historical source. Kept as a lightweight property within the FRBR–FaBiO + FOAF/REL stack; aligned with CIDOC CRM participation modeling.
